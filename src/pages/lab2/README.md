
# Лабраторна робота №2

Задание:Реализовать программу, выполняющую перевод целого неотъемлемого числа из десятичной в двоичную систему счисления. Учтите, что программа должна содержать комментарии к основным блокам и структурам программы, вести диалог с пользователем и корректно обрабатывать ошибки при вводе данных.
